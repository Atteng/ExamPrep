# Exam Prep

Exam Prep is an intelligent self-study platform designed to help students ace their TOEFL, GRE, and German language exams. It leverages Google Gemini AI to generate adaptive practice questions and provide instant feedback.

## Key Features

- **Multi-Exam Support:** Tailored practice for TOEFL iBT, GRE General, and Goethe-Zertifikat.
- **AI-Generated Content:** Infinite practice questions for Reading, Listening, Speaking, and Writing.
- **Real-time Analytics:** Track progress with detailed scoring breakdowns and performance trends.
- **Secure & Fast:** Powered by Supabase for authentication/database and Next.js for a high-performance frontend.
